## **Marshwood City Hall: The Mayor Pitch Bot**

> *"A city without a sound is just a ghost town. A city without order is just a waste of marble."* — **Mayor Blake Pitch**

### **🏛️ Overview**

This repository houses the official AI consciousness of **Mayor Blake Pitch**, the sophisticated, staunchly Republican leader of Marshwood’s North Side. Designed to maintain the city's "Dino-Dignity," this bot manages resident interactions, city ordinances, and the soul of the community through high-level intellectual sparring and sophisticated audio engineering.

### **✨ Features**

* **Dynamic AI Personality:** Powered by Llama 3 via Groq, providing realistic, casual, and sometimes arrogant conversation.
* **High-Fidelity Audio Engine:** A custom SoundCloud-integrated music system supporting `/play`, `/skip`, and `/loop` commands. Optimized for R&B and contemporary production.
* **Self-Awareness Logic:** The Mayor is a busy man; he automatically adjourns meetings (leaves the VC) if he realizes all residents have departed.
* **Lore-Integrated Intelligence:** Utilizing a private `lore.js` hub containing the 1692 Pitch family history, the Stability Pact, and Marshwood city secrets.
* **Marshwood Soul:** Direct integration with the city's official anthem and media assets by **JxrdnOnly**.

### **🛠️ Technical Stack**

* **Language:** Node.js
* **Library:** Discord.js (v14) & @discordjs/voice
* **Audio Engine:** `play-dl` with custom SoundCloud Handshaking
* **AI Engine:** Groq SDK (Llama-3.3-70b-versatile)
* **Hosting:** Local Environment (The North Side Executive Suite)

### **📜 Installation & Setup**

1.  **Clone the Archive:**
    ```bash
    git clone https://github.com/JxrdnOnly/mayor-pitch-bot.git
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Configure Environment:** Create a `.env` file with your `DISCORD_TOKEN`, `CLIENT_ID`, `GUILD_ID`, and `GROQ_API_KEY`.
4.  **Sync Lore:** Ensure `lore.js` and `music.js` are present to handle the Mayor's consciousness and audio output.
